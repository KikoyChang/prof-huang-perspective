## 1. INS REVIEW 与网络社会研究所网站

### 新增判断

INS 网站不只是机构宣传页，而是黄孙权方法的“议题机器”：通过 INS REVIEW、网络社会年会、国际讲座、黑客松、出版、Web3、诸众之貌等栏目，把技术哲学、媒体政治经济学、平台合作主义、数字劳动、全球南方、AI基础设施、媒介考古和艺术教育组织成持续议程。

### 关键证据

1. INS 首页将最新内容分为 INS REVIEW、网络社会年会、国际讲座、黑客松、出版、WEB3、诸众之貌等栏目，并列出大量议题标签：技术哲学、Henri Lefebvre、平台合作主义、网络社会学、黑客松、合作社、许煜、智慧都市网络等。
   - 来源：https://www.caa-ins.org/
   - 类型：机构平台，可信度：高

2. INS REVIEW 栏目包含《一个理论的蓝图：信息主义、网络和网络社会》《后到者必对先进者犯罪--评中美技术冲突》《机器政治--互联网崛起与集权新世纪》《保持思想开放的斗争》《斯蒂格勒对话艾利·杜灵：技术作为记忆》等。
   - 来源：https://caa-ins.org/archives/category/ins%EF%B9%8Dreview
   - 类型：机构栏目，可信度：高

3. INS 关于页明确说，网络社会研究强调 network 而非 internet，关注资讯技术连动的社会文化变革、流动空间和实质空间的互馈生产，并以项目驱动、国际会议、工作坊、理论实验室组织教学研究。
   - 来源：https://www.caa-ins.org/about
   - 类型：机构说明，可信度：高

4. 第四届网络社会年会“网民21：超越个人账户”中，黄孙权以“网络造事（network as initialize）”为入口，把第一届到第四届的年会议题串成实践理论与研究进程，并提出要从个人敏感式创作到创作集体、从独角兽公司到劳动者自组平台。
   - 来源：https://caa-ins.org/archives/6154
   - 类型：机构记录/会议报道，可信度：高

### 对 skill 的深化

- 原模型“教育是组织研究网络”应加强为“议题机器与制度化行动”：黄孙权不是只做课程，而是用栏目、年会、出版、黑客松和平台不断制造公共议题。
- 原模型“网络不是互联网”应扩展：网络社会研究是一套研究基础设施，长期跟踪协议、平台、基础设施、劳动、合作社、AI、全球南方和媒介考古。

## 2. Heterotopias

### 新增判断

Heterotopias 是黄孙权较直接的个人写作/播客/档案出口。它补足了机构文本中较少显现的三点：第一，破报与社会运动档案；第二，Web3/去平台化出版实践；第三，用播客和AI快速组织理论书单的实验性写作方式。

### 关键证据

1. Heterotopias 首页标示为 Huang SunQuan 的 Substack，副标题是“Poetry is puked, not plotted.”，2026年可见仍在更新。
   - 来源：https://news.heterotopias.org/
   - 类型：个人平台，可信度：高

2. 2025年7月1日《STS, SCOT與ANT》说明他决定制作一系列 podcast，用“非人类的方式”快速理解当今网络社会与技术文化需要的信息，并列出 STS、SCOT、ANT、技术社会塑造、技术与社会理论等书目。
   - 来源：https://news.heterotopias.org/p/sts-scotant
   - 类型：个人播客/书单，可信度：高

3. 2023年5月2日《web3 的 substack: Paragraph》把 Paragraph 视为 Web3 与 Substack 的过渡桥梁，关注钱包订阅、Arweave 存储、NFT、ISCN、平台抽成和去平台化出版。
   - 来源：https://news.heterotopias.org/p/web3-substack-paragraph
   - 类型：个人短文，可信度：高

4. 2022年6月5日《89到97--六四民運與非殖民過程中的香港民主運動》重发 1996 年《破报》旧文，并说明发行 NFT 所得将作为破报公共资料库基金。
   - 来源：https://news.heterotopias.org/p/8997
   - 类型：个人档案/旧文再发布，可信度：高

5. 《黑手那卡西群訪》强调黑手那卡西不是为社会运动伴奏，而就是社会运动的一部分，并参与“新人”的制造。
   - 来源：https://news.heterotopias.org/p/8e7/
   - 类型：个人档案/访谈，可信度：高

### 对 skill 的深化

- 增加“档案不是保存过去，而是为未来行动筹资/再流通”的判断。
- 增加“出版基础设施政治”：从微信、WordPress、Substack、Paragraph、Arweave、ISCN、NFT看知识如何被平台化或重新公共化。
- 增加“理论速写/播客化”的表达侧面：他会把复杂书单压缩成可传播的技术文化入门路径。

## 3. 第十届网络社会年会：AI 与模型资本

### 新增判断

第十届网络社会年会“互联网之终？AI（ài）来不来”将黄孙权的网络社会问题推进到 AI 时代：开放协议的互联被模型界面的互导替代，公共知识被编纂为可出租、可封装、可套利的模型资本。

### 关键证据

1. 活动页显示黄孙权为第十届网络社会年会总召集人，议题为“互联网之终？AI（ài）来不来”。
   - 来源：https://www.caa-ins.org/archives/12745
   - 类型：机构活动页，可信度：高

2. 引言指出，早期网际世界以 HTTP、SMTP、RSS、ActivityPub 等开放协议维系互联，而当下 API 条款、系统提示、对齐策略、内容筛检与评测成为新的黑箱协议，形成从互联到互导的转变。
   - 来源：https://www.caa-ins.org/archives/12745
   - 类型：机构活动页，可信度：高

3. 引言将 AI 作为“模型资本”与“一般智能”的公司化圈占来分析，关注语料主权、算法主权、基础设施资源斗争、梯度地租、数据标注劳动和作者性。
   - 来源：https://www.caa-ins.org/archives/12745
   - 类型：机构活动页，可信度：高

### 对 skill 的深化

- 原模型“网络不是互联网”要更新为：从开放协议到模型界面，从互联到互导。
- 决策启发式要加入：看一个 AI 系统的 API/提示/对齐/评测如何成为黑箱协议。
- 最新动态应更新到 2025-2026，而不是停在 2023。
